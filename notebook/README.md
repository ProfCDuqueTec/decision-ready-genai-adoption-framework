# Analysis Notebooks

This folder contains the reusable analysis notebook associated with the **Decision-Ready GenAI Adoption Framework**.

The notebook documents and operationalizes the analytic workflow used to study generative AI adoption in engineering education, with particular relevance for **Global South** and other **resource-heterogeneous contexts**.

## Main notebook

### `Duque_and_Ostos_Educon_2026_definitive.ipynb`

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1PYhS92aJnxb8OCl2GLC__eqFCg45f57i?usp=sharing)

This is the curated and publication-ready notebook for the framework.

It includes:
- authorship and citation information;
- documented analytic sections;
- construct-level descriptives;
- item-level diagnostics;
- HC3-robust OLS analysis;
- multicollinearity diagnostics;
- segmentation outputs;
- usage summaries by activity and semester.

## Purpose

This notebook is intended to support:
- methodological reuse;
- local adaptation of the analytic workflow;
- transparent replication in engineering education contexts;
- low-cost implementation in institutions without heavy analytic infrastructure.

## Expected input

The notebook is designed to work with a survey dataset exported in spreadsheet format.

### Primary expected file

- `respuestas.xlsx`

This file should contain the survey responses required by the workflow.

## Data requirements

The notebook expects a dataset structured around the instrument used in the study, including:
- construct items for **PE, EE, SI, and FC**;
- usage-frequency items associated with **BI**;
- basic contextual variables such as semester, gender, and prior AI use.

Users adapting the workflow should ensure that:
- variable names are aligned with the expected structure;
- item coding is consistent;
- response scales are preserved or carefully documented if modified.

## Recommended accompanying files

To facilitate reuse, this notebook should be used together with the materials in the repository folders:
- `instrument/` for the reusable survey structure;
- `templates/` for data dictionaries, synthetic sample data, and output templates;
- `docs/` for replication, adaptation, and ethics guidance.

## Outputs generated by the notebook

The notebook produces the following main outputs:

### 1. Construct-level descriptives
Summary statistics for:
- Performance Expectancy (PE)
- Effort Expectancy (EE)
- Social Influence (SI)
- Facilitating Conditions (FC)
- Behavioral Intention (BI)

### 2. Item-level descriptives
Descriptive summaries for individual items within each construct.

### 3. HC3-robust OLS regression
Regression estimates for BI with robust standard errors and confidence intervals.

### 4. Diagnostic outputs
- variance inflation factors (VIF)
- heteroskedasticity test output
- residual normality output

### 5. Segmentation summary
Cluster-based profile summary across UTAUT constructs and BI.

### 6. Usage summaries
- usage frequency by academic activity;
- usage frequency by semester and activity.
