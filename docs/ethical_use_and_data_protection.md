# Ethical Use and Data Protection

This repository is intended to support responsible reuse of a methodological framework for studying generative AI adoption in engineering education.

## What is included

This repository includes:
- reusable instrument materials,
- analysis notebooks,
- synthetic sample data,
- variable dictionaries,
- and replication guidance.

## What is not included

This repository does not include:
- participant-level data from the original study;
- identifiable student information;
- institutional records;
- raw survey exports from the original deployment.

## Recommended safeguards for reuse

Users should:
- obtain local ethical approval when required;
- use informed consent;
- minimise collection of direct identifiers;
- avoid unnecessary demographic granularity;
- review re-identification risks before sharing any data.

## Data sharing recommendation

For open repositories, share:
- synthetic datasets,
- empty templates,
- aggregated outputs,
- and code.

Do not share real participant-level data unless:
- ethical approval permits it,
- consent allows it,
- and re-identification risk is acceptably low.

## Alignment principle

Open science should strengthen trust, not weaken participant protection.
