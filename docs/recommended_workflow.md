# Recommended Workflow

## Minimal workflow
If you need a fast local diagnosis:

1. Adapt instrument
2. Collect survey data
3. Compute construct descriptives
4. Inspect item-level asymmetries
5. Run HC3-robust OLS
6. Summarise implications

## Full workflow
For deeper replication:

1. Adapt instrument
2. Secure ethics alignment
3. Collect data
4. Clean and structure dataset
5. Run construct-level analysis
6. Run item-level diagnostics
7. Run robust regression
8. Generate segment profiles
9. Export tables
10. Interpret in relation to local conditions
11. Prepare action recommendations

## Interpretation reminder
The strongest value of this framework is not only measuring attitudes.
It is locating actionable friction.
