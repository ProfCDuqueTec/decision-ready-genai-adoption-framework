# FAQ

## Is this repository meant to replace local research design?
No. It is meant to support adaptation and replication, not to remove the need for local judgement.

## Can I modify the survey wording?
Yes, but preserve the conceptual meaning of each construct.

## Can I use my own academic activity categories?
Yes, as long as you document the change clearly.

## Can I use this outside engineering education?
Yes, with caution. The framework may be transferable, but interpretation should be context-specific.

## Does this repository contain the original study data?
No. It contains only reusable materials, templates, and synthetic data.

## Can I run the notebooks in Google Colab?
Yes. The notebooks are designed to be Colab-friendly.

## How should I cite this work?
Please cite the associated paper and, when appropriate, this repository.
