# Decision-Ready GenAI Adoption Framework

A replication-ready analytic framework for studying generative AI adoption in engineering education, with particular relevance for Global South contexts and other resource-heterogeneous settings.

This repository provides a reusable package built around an adapted UTAUT-based instrument, transparent analysis workflows, and implementation templates designed to support local replication, contextual adaptation, and decision-ready interpretation.

## Purpose

This repository was prepared to support the reuse and adaptation of the methodological framework presented in the study:

**Duque Becerra, C. R., & Ostos Morales, I.**  
*Decision-Ready Analytic Framework for Generative AI Adoption in Engineering: Segment-Aware, Context-Sensitive, Replication-Ready.*

The package is intended for researchers, engineering education scholars, instructional designers, and institutional leaders who wish to:

- study student adoption of generative AI in engineering education;
- identify integration frictions beyond general attitudes;
- apply a transparent, low-cost analytic pipeline;
- generate actionable evidence for local decision-making;
- support replication across Global South and resource-heterogeneous contexts.

## What this repository contains

### 1. Instrument materials
A reusable UTAUT-based survey structure for studying generative AI adoption in academic settings, including construct mapping and administration guidance.

### 2. Analysis notebooks
Jupyter / Colab-compatible notebooks for:
- construct-level descriptives;
- item-level diagnostics;
- HC3-robust OLS analysis;
- segment-based profiling;
- table export workflows.

### 3. Reuse templates
Data templates, table structures, and variable dictionaries to support fast adaptation.

### 4. Replication guidance
Documentation for local adaptation, ethical use, data protection, and practical implementation.

## Intended use

This repository is designed to support **methodological reuse**, not to distribute participant-level research data from the original study.

Users are encouraged to adapt:
- the survey instrument,
- the data structure,
- the analytic workflow,
- and the interpretation logic

to their own institutional, linguistic, disciplinary, and policy contexts.

## Important note on data

No individual-level participant data from the original study are included in this repository.

To protect participant confidentiality and remain aligned with ethical research practice, this package includes:
- synthetic sample data,
- empty templates,
- variable dictionaries,
- and reusable analysis logic.

## Suggested workflow

1. Adapt the survey instrument to your institutional context.
2. Administer the survey under local ethical approval and consent procedures.
3. Structure your dataset following the provided templates.
4. Run the notebooks locally or in Google Colab.
5. Interpret results using the replication and adaptation guides.
6. Report findings with clear reference to contextual conditions.

## Repository structure

- `instrument/` → survey and construct materials
- `notebooks/` → reusable analysis notebooks
- `templates/` → synthetic data and output templates
- `docs/` → replication, adaptation, and ethics guidance
- `assets/` → visual support files, including QR code and framework image

## Reuse principles

This framework is especially suitable for contexts where institutions need:
- transparent analysis,
- low-cost implementation,
- context-sensitive diagnostics,
- and practical guidance for responsible GenAI integration.
