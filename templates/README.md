# Templates

This folder contains reusable templates to support local replication and adaptation.

## Contents

- `sample_data.csv`  
  Synthetic example dataset with the expected structure.

- `data_dictionary.csv`  
  Description of variable names, coding, and meaning.

- `table_template_constructs.csv`  
  Template for construct-level summary tables.

- `table_template_items.csv`  
  Template for item-level descriptives.

- `table_template_regression.csv`  
  Template for regression reporting.

- `table_template_segments.csv`  
  Template for segment-based profile summaries.

## Important note

The sample dataset is synthetic and is included only to demonstrate the workflow.
It does not contain participant-level data from the original study.
