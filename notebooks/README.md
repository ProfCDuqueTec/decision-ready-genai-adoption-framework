# Analysis Notebooks

This folder contains reusable notebooks for the analytic workflow associated with the decision-ready GenAI adoption framework.

## Recommended sequence

### 01_construct_descriptives.ipynb
Compute construct-level descriptives and confidence intervals.

### 02_item_level_diagnostics.ipynb
Inspect item-level asymmetries within constructs.

### 03_hc3_robust_ols.ipynb
Estimate OLS models with HC3-robust standard errors.

### 04_segmentation_profiles.ipynb
Generate segment-based profiles using standardized construct scores.

### 05_export_tables.ipynb
Export publication-ready tables and summary outputs.

## Running options

These notebooks may be run:
- locally in Jupyter,
- or in Google Colab.

## Data note

The notebooks are designed to work with:
- synthetic sample data for demonstration,
- or locally supplied institutional datasets following the template format.
